# 🏠 Property Portal - Real Estate Analysis Platform

A complete real estate analysis platform with AI-powered property value predictions and comprehensive market insights.

## 🎯 Features

### Property Estimator Module
- **Instant Price Predictions** - Get AI-powered property value estimates
- **Property Comparison** - Compare multiple properties side-by-side
- **Interactive Charts** - Visualize price comparisons with bar charts
- **Input Validation** - Real-time validation with range warnings
- **Historical Tracking** - View your prediction history

### Market Analysis Module
- **Market Statistics** - View average, median, min, and max property prices
- **Price by Bedrooms** - Bar charts showing average prices for different bedroom counts
- **Market Distribution** - Pie charts showing price tier distribution
- **Feature Correlations** - Understand which features impact price the most
- **What-If Analysis Tool** - See how changing these affects property prices:
  - Square Footage
  - Number of Bedrooms
  - School Rating
- **Data Export** - Export analysis results in CSV and PDF formats

## 🛠 Tech Stack

- **Frontend**: Next.js, Tailwind CSS, Recharts
- **Backend**: Python, FastAPI
- **ML Model**: scikit-learn
- **Data Processing**: Pandas, NumPy

## 📋 Prerequisites

- Python 3.12+
- Node.js 20+
- npm
